# Sciolla Sistemas Digitales - Landing Page

Landing page profesional para Sciolla Sistemas Digitales con formulario de contacto integrado con Formspree.

## Estructura del Proyecto

```
/
├── index.html          # Archivo principal HTML
├── styles.css          # Estilos CSS
├── script.js           # JavaScript funcional
├── logo.png            # Logo de la empresa
└── README.md           # Este archivo
```

## Características

- ✅ Diseño responsivo (mobile-first)
- ✅ Formulario de contacto integrado con Formspree
- ✅ Botón "Volver arriba" automático
- ✅ Navegación sticky
- ✅ Secciones: Hero, Servicios, Presencia Regional, Presupuesto, CTA
- ✅ Colores corporativos: Azul marino (#001F3F) y Cian (#00D4FF)
- ✅ Tipografía: Poppins (títulos) e Inter (cuerpo)

## Despliegue en Netlify

1. Conecta este repositorio a Netlify
2. Configura la rama principal como rama de despliegue
3. Netlify detectará automáticamente el `index.html` como archivo principal
4. El sitio se desplegará en: `https://tu-dominio.netlify.app`
